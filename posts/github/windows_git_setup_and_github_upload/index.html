Git 环境配置（Windows）

零基础、小白都能学会

一、注册 GitHub 账号

注册网址： Join GitHub · GitHub

二、下载 Git 客户端安装包

官方下载：Git - Downloading Package

腾讯下载：【Git下载】2023年最新官方正式版Git免费下载 - 腾讯软件中心官网

三、Git 全局配置

安装包安装完后，就可以在桌面右键
点击 Git Bash Here 选项
就可以开始输入命令了

  1. 输入配置账号命令
1

git config --global user.name "你的账号"
  1. 输入配置邮箱命令
1

git config --global user.email "你的邮箱地址"
  1. 输入生成 SSH 公钥命令，然后连续按三次回车就行
1

ssh-keygen -t rsa -C "你的邮箱地址"
  1. 输入查看公钥字符串命令，然后复制字符串到剪切板
1

cat ~/.ssh/id_rsa.pub

四、GitHub 配置 SSH Key

  1. 打开创建SSH Key页面：Add new SSH key
  2. Title 随便填什么都行
  3. Key 就复制之前查看到的公钥字符串粘贴上去就行

五、创建远程仓库

  1. 打开新建仓库页面：New repository
  2. Repository name 填：123
  3. Description 随便填什么都行
  4. 勾选 Public 选项
  5. 其他选项默认就行，不用动
  6. 点击 Create repository 按钮创建

六、上传本地代码到远程仓库

在桌面上创建一个 123 文件夹，并进入该文件夹
右键点击 Git Bash Here 选项
就可以开始输入命令了

  1. 输入创建一个自述文件命令
1

echo "# 123" >> README.md
  1. 输入初始化一个本地仓库命令
1

git init
  1. 输入将刚才的自述文件添加到缓存区命令
1

git add README.md
  1. 输入将缓存区的文件提交到仓库永久存储命令
1

git commit -m "第一次提交"
  1. 输入切换到主支干命令
1

git branch -M master
  1. 输入绑定到远程仓库地址命令

记得一定要用 SSH 的，不要用 HTTPS，
否则等下上传的时候会一直弹账号密码输入框，
GitHub 现在不允许用账号密码提交了，
就算输入正确的账号密码也会一直报错。

1

git remote add origin git@github.com:你的账号/123.git
  1. 输入上传代码命令
1

git push -u origin master
  1. 输出这个提示则表示成功了！

To GitHub.com:SMWHff/123.git

  • [new branch] master -> master

Branch ‘master’ set up to track remote branch ‘master’ from ‘origin’.
